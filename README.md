# UUV-simulation-with-ROS-Gazebo-and-Simulink
Co-simulation of an Unmanned Underwater Vehicle (UUV) using ROS/Gazebo and Simulink. The simulation verifies an MRAC-PID controller with an anti-windup mechanism for robust trajectory tracking. Control commands are generated in Simulink and applied to the Gazebo UUV model via ROS communication.
